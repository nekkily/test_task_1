Единственный файл с настройками - db.php (файл с подключением к базе данных).

Инструкция: 

1.Залить код сайта на сервер (без database.sql).
2.Создать базу данных с названием comments.
3.Импортировать в созданную базу database.sql (с помощью консоли или phpmyadmin).
4.В файле db.php прописать свои данные для доступа к базе ($db_host, $db_username, $db_password).
Готово!

Если на экране появляется ошибка: Warning: mysqli::__construct(): (HY000/1045), проверьте данные для подключения к бд.
