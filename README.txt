Единственный файл с настройками - db.php (файл с подключение к базе данных).

Инструкция: 

1.Залить код сайта на сервер (без database.sql).
2.Импортировать базу данных database.sql (с помощью консоли или phpmyadmin).
3.В файле db.php прописать свои данные для доступа к базе ($db_host, $db_username, $db_password).
Готово!
